Description: This request retrieves F5 Configurations from the specified LTM pair for specified VIP addresses. With the modify key set to "false", the existing configurations are returned as they exist along with a JSON summary of the VIP and pool information. This JSON summary can be updated with new information that allows you to set the modify flag to "true" which will retrieve the configurations, update them with the provided information, and return the modified configurations. This tool does not have the ability to copy SSL certificates, keys, or chains as part of the process. Upon receiving the JSON summary from this tool, ensure that the reported SSL Profile information exists on the new LTMs that on which the configuration will be placed before applying the configurations. By setting any value for collectionName, a JSON file will be generated on the S: drive that can be imported into Postman as a collection.

JSON Data:
{
"APIKey": "",
"sourceLTMs": ["10.90.8.171", "10.90.8.172"],
"vips": ["10.204.67.10", "10.204.67.11"],
"collectionName": "",<
"newLTM": ["10.90.8.131"],
"modify": false,
"modifyData": {}
}
